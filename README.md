# 卫戍协议 (Garrison Protocol)

明日方舟「卫戍协议」玩法的高保真复刻——融合肉鸽策略、塔防布阵与自走棋战斗的 Python 回合制策略游戏。

## 截图

> 运行 `python -m game.main --arcade` 体验

## 核心玩法

- **回合制决策** — 购买干员、部署布阵、管理资源、升级调度中心
- **盟约系统** — 20 种盟约（8 核心阵营 + 12 特殊主题），通过干员特质叠层，解锁递增加成
- **塔防战斗** — 敌人生成于红门沿路径推进，近战干员阻挡，远程/治疗干员支援，全自动结算
- **晋升与奖励** — 3 张同名干员合成为精锐，附带免费晋升奖励
- **拖拽部署** — 从整备区拖拽干员到地图格子，再拖拽选择朝向

## 快速开始

### 安装

```bash
pip install arcade
```

### 运行

```bash
python -m game.main --arcade
```

设置界面选择策略和难度后点击「开始模拟」。

### 操作

| 操作 | 方式 |
|------|------|
| 购买干员 | 点击商店卡片 |
| 部署干员 | 拖拽整备区干员到地图格子，再拖拽选择朝向 |
| 撤退/出售 | 右键地图上干员 / 右键整备区干员 |
| 刷新商店 | 点击刷新按钮 或 按 R |
| 冻结商店 | 点击冻结按钮 |
| 开始战斗 | 点击右下角按钮 |
| 调整倍速 | 战斗中点击 1x/2x/5x/8x 或按空格 |
| ESC | 取消选择 / 取消拖拽 |

## 盟约一览

### 核心阵营（需 3 人激活）

| 盟约 | 效果 |
|------|------|
| 炎 | 攻击+(23+0.9×层数)%, 生命+一半 |
| 卡西米尔 | 攻击+(50+层数)%, 生命+一半 |
| 维多利亚 | 攻击+(25+0.8×层数)%, 生命+一半 |
| 谢拉格 | 伤害+25%+每层 1%, 生命+一半 |
| 拉特兰 | 攻速+30, 攻击+(20+0.5×层数)%, 生命+一半 |
| 阿戈尔 | 生命+(35+层数)%, 攻击+生命加成的 3/4 |
| 叙拉古 | 攻速+50, 攻击+(15+0.4×层数)%, 生命+一半 |
| 萨尔贡 | 攻击+(50+层数)%, 攻速+50, 生命+一半 |

### 特殊主题（需 1 人激活）

| 盟约 | 效果 |
|------|------|
| 精准 | 攻击+(10+1.2×层数)%, 生命+一半 |
| 灵巧 | 攻速+(15+层数), 再部署-30%; ≥40 层额外+50 攻速 |
| 奥术 | 攻击+(20+层数)% |
| 坚守 | 生命+(25+1.2×层数)% |
| 助力 | 防御+(15+1.2×层数)%, 再部署-30% |
| 突袭 | 攻击+(25+层数)%, 生命+(25+层数)%, 再部署-(25+0.2×层数)%; ≥50 层攻速+50 |
| 不屈 | 被击倒时(18+0.4×层数)%概率半血复活 |
| 独行 | 攻命+60%; 场上超过 1 名独行干员时失效 |
| 远见 | 每 10 层每回合+2 资金; 刷新时(18+0.3×层数)%概率免费; 整备区计入激活 |
| 投资人 | 获得时效果触发 2 次; ≥100 层触发 3 次; 整备区计入激活 |
| 调和 | 核心盟约激活所需在场干员数-1 |

## 项目结构

```
sbq/
├── game/
│   ├── main.py          # 入口
│   ├── arcade_ui.py     # Arcade 渲染、输入、主循环
│   ├── data.py          # 干员、盟约、策略、敌人数据
│   ├── models.py        # 数据模型
│   ├── engine.py        # 游戏状态、回合逻辑、盟约引擎
│   ├── logic.py         # 战斗数值计算
│   ├── map_gen.py       # 随机地图生成
│   └── assets/          # BGM、音效、字体
└── README.md
```

## 技术栈

- **Python 3.10+**
- **Arcade 3.x** — 2D 游戏框架
- 纯 Python，无其他依赖
